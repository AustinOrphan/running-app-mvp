# Running Tracker MVP - Project Purpose

## Overview

A full-stack web application for tracking running activities, built as an MVP (Minimum Viable Product).

## Main Features

- **User Authentication**: Registration and login with JWT and bcrypt
- **Run Tracking**: Create, view, edit, and delete running activities
- **Goals Management**: Set and track running goals with templates and analytics
- **Statistics & Analytics**: View personal records, trends, and insights
- **Achievement System**: Goal completion notifications and streak tracking
- **Responsive Design**: Mobile-friendly interface with swipe navigation

## API Features

- RESTful API for runs, goals, stats, and authentication
- Protected routes with JWT middleware
- User data isolation by user ID
- Sample data generation for testing

## Target Users

Runners who want to track their activities, set goals, and monitor their progress over time.

## Technology Philosophy

- Modern TypeScript-first development
- Component-based React architecture
- Type-safe backend with Prisma ORM
- Comprehensive testing strategy (unit, integration, e2e, accessibility)
- Clean, maintainable code structure
